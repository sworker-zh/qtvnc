# 移植 qtvnc 到其他 qmake 项目

## 第1步：复制文件

将以下文件复制到你的项目中（建议放在 `3rdparty/qtvnc/` 下）：

```
3rdparty/qtvnc/
  connectdialog.h
  connectdialog.cpp
  vncconnection.h
  vncconnection.cpp
  vncviewerwidget.h
  vncviewerwidget.cpp
```

`main.cpp` 不需要复制。

## 第2步：准备 libvncclient

需要两样东西：**头文件**和**编译好的库文件**。

从当前项目的 `build/` 目录获取：

```
# 头文件（两个目录都要）
include/rfb/rfbclient.h
include/rfb/rfbproto.h
include/rfb/keysym.h
include/rfb/threading.h
include/rfb/rfbregion.h
build/include/rfb/rfbconfig.h    # CMake 生成的配置头

# 库文件
build/libvncclient.a
```

复制到你的项目中：

```
3rdparty/libvnc/
  include/rfb/         ← 放所有 .h
  lib/libvncclient.a   ← 放编译好的静态库
```

## 第3步：在你的 .pro 文件中添加

```pro
# === VNC Viewer 模块 ===

# 头文件搜索路径
INCLUDEPATH += $$PWD/3rdparty/qtvnc \
               $$PWD/3rdparty/libvnc/include

# 添加 VNC viewer 源文件
SOURCES += \
    $$PWD/3rdparty/qtvnc/vncconnection.cpp \
    $$PWD/3rdparty/qtvnc/connectdialog.cpp \
    $$PWD/3rdparty/qtvnc/vncviewerwidget.cpp

HEADERS += \
    $$PWD/3rdparty/qtvnc/vncconnection.h \
    $$PWD/3rdparty/qtvnc/connectdialog.h \
    $$PWD/3rdparty/qtvnc/vncviewerwidget.h

# 链接 libvncclient + winsock
LIBS += -L$$PWD/3rdparty/libvnc/lib -lvncclient -lws2_32
```

## 第4步：在你的代码中使用

```cpp
#include "connectdialog.h"
#include "vncviewerwidget.h"

// 方式1：独立对话框流程（和 qtvnc 一样）
ConnectDialog dlg(this);
if (dlg.exec() == QDialog::Accepted) {
    VncViewerWidget *viewer = new VncViewerWidget(this);
    viewer->startConnection(dlg.host(), dlg.port(), dlg.password());
    viewer->show();
}

// 方式2：直接嵌入到你的窗口中（不弹对话框）
VncViewerWidget *vnc = new VncViewerWidget(this);
// 嵌入到布局里
someLayout->addWidget(vnc);
// 直连，不用对话框
vnc->startConnection("192.168.1.100", 5900, "password");

// 监听断开事件
connect(vnc, &VncViewerWidget::connectionLost, this, [this]() {
    qWarning() << "VNC disconnected";
});
```

## 注意事项

- **跨平台**：Linux/macOS 上不需要 `-lws2_32`，可按平台条件添加：
  ```pro
  win32: LIBS += -lws2_32
  ```
- **静态库需同编译器**：`libvncclient.a` 必须用与目标项目相同的编译器（如都是 MinGW 64-bit）构建，否则链接会出错
- **Qt 版本**：qtvnc 代码兼容 Qt5.12+ 和 Qt6，无需修改
